# Editor Layout Redesign

## تحديث: تصميم محرر المقالات

تم إعادة تصميم صفحة إنشاء وتحرير المقالات بالكامل لتحسين تجربة المستخدم.

## التغييرات الرئيسية

### قبل (Before):
- **تصميم عمودين**: محرر كبير على اليمين + شريط جانبي على اليسار
- المحرر يأخذ مساحة كبيرة جداً
- خيارات النشر مبعثرة في بطاقات منفصلة في الشريط الجانبي
- صعوبة رؤية جميع الخيارات في نفس الوقت

### بعد (After):
- **تصميم عمود واحد**: كل شيء في مكان واحد
- محرر أصغر وأكثر تركيزاً (300-400px ارتفاع)
- خيارات النشر معروضة بشكل مدمج أسفل المحرر
- تنظيم أفضل في شبكة (grid) متجاوبة
- جميع الخيارات مرئية دون الحاجة للتمرير كثيراً

## CSS Classes الجديدة

### Layout Classes:
- `.article-editor-form-single` - الحاوية الرئيسية للنموذج
- `.editor-container-compact` - حاوية المحرر المصغرة
- `.publish-options-inline` - حاوية خيارات النشر المدمجة

### Section Classes:
- `.publish-section` - قسم من خيارات النشر
- `.section-label` - عنوان القسم
- `.options-grid` - شبكة الخيارات (responsive grid)

### Form Element Classes:
- `.option-item` - عنصر خيار واحد
- `.option-item-full` - عنصر يأخذ عرض كامل
- `.checkboxes-row` - صف من checkboxes
- `.checkbox-label-inline` - تسمية checkbox مدمجة
- `.input-select-inline` - قائمة منسدلة مدمجة
- `.input-text-inline` - حقل نص مدمج
- `.input-textarea-inline` - منطقة نص مدمجة

### Image & Upload Classes:
- `.image-preview-inline` - معاينة الصورة المدمجة
- `.btn-remove-inline` - زر حذف الصورة
- `.file-upload-label-inline` - تسمية رفع الملف

### Button Classes:
- `.btn-publish-main` - زر النشر الرئيسي

### Helper Classes:
- `.help-text` - نص مساعد صغير

## Responsive Breakpoints

```css
/* Desktop: 769px+ */
- شبكة 3 أعمدة (auto-fit, minmax(280px, 1fr))
- زر النشر: 300px عرض

/* Tablet: 481-768px */
- شبكة عمود واحد
- محرر: 250-350px ارتفاع
- زر النشر: عرض كامل

/* Mobile: ≤480px */
- تباعد أصغر (padding)
- checkboxes عمودية
- جميع العناصر عرض كامل
```
