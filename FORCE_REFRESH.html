🔄 Cache Clear Guide

⚠️ Important: Your browser is showing cached (old) content. The new video section IS deployed, but your browser hasn't loaded it yet.

✅ Quick Fix - Choose One:

Option 1: Hard Refresh (Recommended)

Windows/Linux: Press Ctrl + Shift + R or Ctrl + F5

Mac: Press Cmd + Shift + R

Option 2: Clear Cache via DevTools

  1. Press F12 to open DevTools
  2. Right-click the browser refresh button
  3. Select "Empty Cache and Hard Reload"

Option 3: Private/Incognito Window

Open your website in a new private/incognito window - this always loads fresh content

Windows: Ctrl + Shift + N (Chrome) or Ctrl + Shift + P (Firefox)

🔄 Go to Site & Force Reload 🆕 Open in New Tab

🎯 What to Look For:

After clearing cache, scroll down past "Top Stories". You should see:

  • ✅ A black section with "شاهد" (Watch) heading
  • ✅ Videos displayed in a horizontal carousel
  • ✅ Play button overlays on video thumbnails
  • ✅ Navigation arrows (← →) on desktop
  • ✅ Dark background (#1a1a1a)

🐛 Still Not Working?

  1. Open DevTools (F12) → Console tab
  2. Check for any red error messages
  3. Go to Network tab → refresh → check if index.html is loading (Status 200)
  4. Verify the Response shows recent content (check HTML source)
